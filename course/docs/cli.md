# Commands for Linux Bash
## Hide Files

```sh
 ls -a
 ```

 ## Do empty document

```sh
touch name.txt
 ```


 ## Remove
```sh
rm
 ```


## Create directory
```sh
 mkdir
 ```


 ## Recursive

```sh
 -f
 ```
